# Kuaiapp/DB

Swoole\Coroutine\MySQL 的 PDO接口实现

## Install

Install by [Composer](https://getcomposer.org)  

```bash
composer require kuaiapp/db
```

## Require

- Swoole > 2.1.2, enable coroutine, enable mysqld
